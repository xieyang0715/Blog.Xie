# 编辑命令

```bash
ctrl + a 跳转至行首
ctrl + u 删除至行首

ctrl + e 跳转至行尾
ctrl + k 删除至行尾
```

```bash
ctrl + xx 行首往返跳转

alt  + b  向前移动一个单词
alt  + f 向后移动一个单词

ctrl + w 向前删除至单词首
alt  + d 向后删除至单词尾

ctrl + y 粘贴
```



重新执行命令

```bash
ctrl + r 搜索命令历史
alt + . 使用上一条命令的最后一个参数, 可以反复按.取所有历史的参数。
```
