折半

```python
val = input('>>> ')
val = int(val)
if val >= 1000: # fold
    if val >= 10000:
        print(5)
    else:
        print(4)
else:
    if val >= 100:
        print(3)
    elif val >= 10:
        print(2)
    else:
        print(1)
```
